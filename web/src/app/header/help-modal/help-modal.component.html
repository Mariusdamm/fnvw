Getting Started

Gruppen

In einer Gruppe können Einträge gegliedert und zusammengefasst werden. Beispielsweise kann man eine Gruppe "Lebensmittel" erstellen und in dieser Gruppe alle Einkäufe in Lebensmittelgeschäften zusammenfassen.

Um eine Gruppe zu erstellen, muss man auf das Gruppen-Icon klicken. Das Icon im grünen Bereich erstellt neue Einnahme-Gruppen, das im roten Bereich neue Ausgabe-Gruppen.

Einträge

Um Einträge hinzuzufügen, muss zuerst eine Gruppe erstellt werden. Anschließend klickt man auf das Eintrag-Icon der entsprechenden Gruppe und gibt einen Namen des Eintrags und den Betrag ein.

Monate

Um einen neuen Monat zur Übersicht hinzuzufügen, muss auf das Monats-Icon unten rechts geklickt werden. Anschließend muss der jeweilige Monat ausgewählt werden.

Es können nur Monate hinzugefügt werden, für die bereits Einträge existieren.
