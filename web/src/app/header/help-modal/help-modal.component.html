Getting Started

Gruppen

Um Gruppen hinzuzufügen, muss zuerst auf das Plus-Icon unten rechts und dann auf das Gruppen-Icon geklickt werden. Anschließend muss man alle Daten eingeben und über den Button bestätigen.

Gruppen dienen dazu, Einnahmen und Ausgaben zu gruppieren. Beispielsweise kann man eine Gruppe für variable Ausgaben und für feste Ausgaben machen.

Kategorien

Um Kategorien hinzuzufügen, müssen zuerst Gruppen erstellt werden. Anschließend muss zuerst auf das Plus-Icon unten rechts und dann auf das Kategorie-Icon geklickt werden. Zuletzt muss man alle Daten eingeben und über den Button bestätigen.

Kategorien dienen dazu, Einnahmen und Ausgaben innerhalb Gruppen zu gruppieren. Beispielsweise kann man in einer Gruppe für variable Ausgaben die Kategorien Lebensmittel, Kosmetik und Tanken erstellen. So kann man die Einnahmen und ausgaben pro Kategorie besser im Blick behalten

Einträge

Um Einträge hinzuzufügen, müssen zuerst Gruppen und Kategorien erstellt werden. Anschließend muss zuerst auf das Plus-Icon unten rechts und dann auf das Eintrag-Icon geklickt werden. Zuletzt muss man alle Daten eingeben und über den Button bestätigen.

Einträge stellen einzelne Beträge dar. Sie werden einer Kategorie innerhalb einer Gruppe zugeordnet. In der Gruppe für variable Ausgaben und der Kategorie "Lebensmittel" wäre es denkbar für jedes Mal, wenn man Lebensmittel einkauft, einen Eintrag mit dem Namen des Lebensmittelgeschäfts zu erstellen.

Monate

Um einen neuen Monat zur Übersicht hinzuzufügen, muss auf das Plus-Icon und dann auf das Monats-Icon geklickt werden. Anschließend muss der jeweilige Monat ausgewählt werden.

Es können nur Monate hinzugefügt werden, für die bereits Einträge existieren.
